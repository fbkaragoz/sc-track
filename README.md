# SC Tracks

A personal SuperCollider music production project.

## Overview
This project contains SuperCollider code and compositions for experimental audio synthesis and sound design.

## Requirements
- SuperCollider 3.x or later

## Installation
1. Clone or download this repository
2. Open SuperCollider
3. Navigate to the project directory
4. Execute the main `.scd` files

## Project Structure
```
sc_tracks/
├── README.md
├── synths/          # Synth definitions
├── compositions/    # Musical compositions
└── samples/         # Audio samples
```

Currently I am only dumping the initial structure so the above tree will appear as I work on the project.

## Usage
Start SuperCollider and evaluate code blocks to generate sounds and compositions.

If you are using vscode, you can enable syntax highlights by installing grammar extension and enabling on JSON.
